== Description ==

Connects Arcavis with a WooCommerce Webshop. Articles, Groups, Sales, Tags are being synchronized to WooCommerce while WooCommerce will send back Customers and Orders to Arcavis.

== Installation ==

1. Download and Activate the Plugin in your WooCommerce WordPress Installation
2. Go to Settings -> Arcavis Settings and Enter your Arcavis Credentials
3. Hit Save and wait for all articles to be synchronized
